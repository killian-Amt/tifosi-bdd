# Projet Base de Données – Le Tifosi

## Description

Ce projet consiste à concevoir et implémenter une base de données relationnelle MySQL pour le restaurant de street-food italien **Le Tifosi**.

La base de données permet de gérer :

- Les focaccias
- Les ingrédients
- Les menus
- Les boissons
- Les marques
- Les relations entre ces éléments

Le projet respecte le modèle conceptuel fourni dans le cahier des charges.

---

## Technologies utilisées

- MySQL
- SQL
- MySQL Workbench
- GitHub
